Emotion Recognition using In-game Parameters (ERIP)

ERIP — это проект по распознаванию эмоций на основе параметров, полученных из игровой среды.
Проект включает скрипты для подготовки данных, обработки изображений и обучения модели для предсказания эмоционального состояния игрока на основе входных параметров.

🧠 Описание

Цель ERIP — реализовать систему, которая:

загружает и обрабатывает данные, связанные с игровыми параметрами (включая изображения и числовые признаки),

подготавливает датасет для обучения,

обучает модель для распознавания эмоционального состояния,

оценивает её на тестовых данных.

Проект написан с использованием Python, содержит утилиты для предобработки данных и примерные заготовки для дальнейшего машинного обучения и визуализации результатов.

🚀 Быстрый старт
📦 Требования

Убедись, что у тебя установлен Python (рекомендуется 3.7+). Рекомендуется создать виртуальное окружение:

python -m venv venv
source venv/bin/activate   # Mac / Linux
venv\Scripts\activate      # Windows
📥 Установка зависимостей

Создай requirements.txt (если ещё нет) и пропиши зависимости, например:

numpy
pandas
opencv-python
scikit-learn
torch
tensorflow
matplotlib

Установка:

pip install -r requirements.txt
💻 Использование
Подготовка данных

Перед началом обучения нужно выполнить предобработку:

python Python/image_preparation.py
python Python/data_convertation.py

Если датасет разбит на части, сначала объединяем:

python Python/data_joining.py
🧪 Обучение модели

Добавь свой скрипт обучения — например:

python train.py

где train.py загружает датасет из erip_dataset.py, строит модель и обучает её.

Пример:

from erip_dataset import ERIPDataset
from model import EmotionNet

dataset = ERIPDataset("DATASET.csv")
model = EmotionNet()
model.train(dataset)

Этот блок нужно адаптировать под вашу архитектуру модели.
